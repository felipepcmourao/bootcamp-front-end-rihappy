Seletores CSS

Bem-vindo!

No CSS nós possuímos seletores por:

  • Por Tipo: Busca elementos por uma tag HTML específica.
  • Por ID: Busca elementos através de um atributo "id"
  • Por Classe: Busca elementos através de um atributo "classe"
  • Universal: Seleciona todos os elementos HTML
  • Por Atributo [atrib]: Seleciona elementos que possuem um atributo específico em nosso documento HTML.
